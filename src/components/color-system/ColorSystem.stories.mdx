# Color System

A token-based color system supporting light/dark themes and WCAG-compliant contrast ratios.

## Naming Conventions

- **Primary, Secondary, Accent**: Brand and accent colors for main actions and highlights.
- **Success, Warning, Error, Info**: Semantic colors for status and feedback.
- **Neutral**: Greys and backgrounds for surfaces, borders, and text.
- **Number Suffixes**: Indicate shade (e.g., `-100` is lightest, `-900` is darkest).

## Best Practices

- Use semantic tokens for status and feedback (e.g., `--color-success-600` for success messages).
- Ensure text and interactive elements meet [WCAG AA](https://www.w3.org/WAI/WCAG21/quickref/) contrast requirements.
- Test color combinations in both light and dark themes.
- Reference tokens via CSS variables for themeability.
